# MeetZY – Find New Friends Locally

MeetZY is a full-stack social networking web application that helps users discover and connect with new friends locally based on shared interests and location.

## 🌟 Features

* User Registration & Login Authentication
* Interest-Based Friend Matching
* Location-Based User Discovery
* Friend Request System
* Real-Time Style Chat Interface
* Profile Management
* Admin Panel
* Responsive UI Design
* AWS EC2 Deployment

---

## 🛠 Technologies Used

### Frontend

* HTML
* CSS
* JavaScript
* React

### Backend

* Python
* FastAPI

### Database

* SQLite / PostgreSQL

### Deployment

* AWS EC2

---

## 📌 Project Objective

The main objective of MeetZY is to help users build meaningful local friendships through shared interests and nearby location matching.

---

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/MeetZY.git
cd MeetZY
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Backend Server

```bash
uvicorn app.main:app --reload
```

### 4. Open in Browser

```bash
http://127.0.0.1:8000
```
